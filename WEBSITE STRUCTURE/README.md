# HOOP Marketing — Frontend

This repo contains the React + Vite frontend for the HOOP Marketing website.

Quick start
1. git clone <repo>
2. npm install
3. npm run dev

Build
- npm run build
- npm run preview

Deploy
- For GitHub Pages, set VITE_BASE_PATH to "/your-repo/" (if not root) and run `npm run build`, then publish the `dist/` folder.
- Alternatively deploy to Vercel/Netlify (no base changes required for root).
